# w

a no-backspace note-taking app.

## what it does

- you can only type forward, no deleting
- blocks backspace, delete, undo, and cut
- auto-saves to browser storage
- dark/light mode toggle

## why

forces you to write without second-guessing. no editing, no perfectionism, just writing.

## usage

open `w.html` in a browser and start typing.

click "clear" to erase everything and start fresh.
